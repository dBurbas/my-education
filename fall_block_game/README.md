## 📝 About

Project is made based on Lab #3 PPOIS BSUIR.
> ### Lab work task:
> General requirements:
> 1. Use the selected library and event-driven programming as a basis for building a graphical application
> 2. Store all possible application settings, such as level maps and entity characteristics, in external files (configurations) in json, yaml, or xml format
> 3. When the application is launched, the user is presented with a menu consisting of the following commands: "start game," "high score table," "help," and "exit." When you select "help", the user is shown information about the game's rules
> 4. The game should have sounds and music (background music)
> 5. When the game is completed, if the game's result is higher than the first line in the high score table, a dialog box will appear with a congratulation and the option to enter a name and save it in the high score table.
> 6. The game should have animations, such as death animations and bullet animations
> 7. Document the implementation using the markdown markup language
> 8. The program's source code and documentation should be hosted on github
>
> Variant: FallBlock
> 1. Implement rotations of geometric shapes
