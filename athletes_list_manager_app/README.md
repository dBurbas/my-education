## 📝 About
> Project is in development 🧱

Project was made based on Lab #2 PPOIS BSUIR.
> ### Lab work task:
> Develop a window application with one main window and several child
> dialogs. Dialogs are called through the corresponding menu items.
> Menu commands should be duplicated on the toolbar.
> ***
> Variant:
> 
> Attribute values:
> - Team type: main/reserve/“n/a”.
> - Position: depending on the sport.
> - Titles: a certain number.
> - Rank: 1st junior, 2nd rank, 3rd rank, KMS, Master of Sports.
>
> Table:
> |Athlete's name| Team Type (if available)| Position| Titles| Sport| Rank|
> |---|---|---|---|---|---|
> | ... | ... | ... | ... | ... | ... |
> Search and deletion conditions:
> - by name or sport;
> - by the number of titles won (the upper and lower limits must be specified when the > task is set);
> - by name or rank
> 
> Note: The list of sports and the list of ranks in the search dialog are collected by the system and displayed in the drop-down list;
